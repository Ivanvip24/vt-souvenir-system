# Brand Identity

> AXKAN visual system, voice guidelines, and decision framework.

## What it does

Defines everything about the AXKAN brand — colors, typography, logo rules, voice, and the "AXKAN Test" for making decisions. This is a reference module, not a running service.

## The AXKAN Test

Apply to every brand decision:
1. Does it trigger pride?
2. Is it culturally authentic?
3. Is it premium?
4. Is it accessible?
5. Will it last?
