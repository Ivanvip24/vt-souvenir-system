# CLAUDE.md - AXKAN Skill Package

Este archivo configura a Claude Code para trabajar con la marca AXKAN.

## Skills Disponibles

Este paquete contiene el skill `axkan` - la identidad completa de marca.

## Instrucciones de Contexto

Cuando trabajes en cualquier tarea relacionada con AXKAN:

1. **Siempre consulta el skill** de AXKAN antes de crear contenido de marca
2. **Aplica el AXKAN Test** a cualquier decisión de diseño o contenido
3. **Usa la paleta de colores oficial** - nunca inventes colores
4. **Mantén la voz de marca** - fresca, aventurera, orgullosa pero accesible

## Logo Rules
