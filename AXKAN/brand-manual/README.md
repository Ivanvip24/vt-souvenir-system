## Key Visual Rules

1. **Decorative AXKAN** = Logo use only
2. **Destination names** = Plain RL AQVA, solid colors per letter
3. **Photography + Illustration** = Illustration frames photo, never competes
4. **Diagonal compositions** = 45° angle for dynamic layouts
5. **Pattern system** = Fixed compositions, spiral mandatory

## Product Design Elements

From TEST series, product magnets include:
- Cloud/scalloped shape silhouette
- MDF wood edge (tan #D4A574)
- Illustrated tropical foliage base
- Regional fauna (destination-specific)
- Colorful destination name with Mayan pattern textures
- Waterfall/water elements where appropriate
- 3D raised letters
