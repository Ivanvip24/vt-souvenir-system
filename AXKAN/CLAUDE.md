### Color Rules
- **60/30/10**: 60% white/neutral, 30% pink, 10% accents
- **Forbidden**: Pink+Red touching, Orange text on Pink, all 5 colors equal weight
- **Safe combos**: Pink+Cyan, Pink+Green, Pink+White+one accent

## Logo Rules

- **Decorative textured "AXKAN" = LOGO USE ONLY**
- Destination names (YUCATAN, CANCUN) use plain RL AQVA with solid colors per letter
- Clear space = X-height on all sides
- Minimum: 30mm print / 120px digital (full logo)
- Jaguar alone: OK for icons, watermarks
- Individual decorative letters: NEVER

## Pattern System

- **Fixed compositions only** - no custom arrangements
- Mandatory element: Spiral (caracol) in every pattern
- Densities: Dense (80%+), Medium (40-60%), Sparse (10-20%)

## Photo vs Illustration

**Rule:** Illustration should FRAME or ACCENT photography, never compete.

- **Photography:** Destinations, products, lifestyle, realistic animals
- **Illustration:** Logo, patterns, icons, decorative frames

**Layer hierarchy:** Photo (back) → Pattern at 10-20% opacity (mid) → Logo/text (front)

## The AXKAN Test

Apply to ALL brand decisions:
1. Does it trigger pride?
2. Is it culturally authentic?
3. Is it premium?
4. Is it accessible?
5. Will it last?

**If any answer is "no," the decision fails.**
