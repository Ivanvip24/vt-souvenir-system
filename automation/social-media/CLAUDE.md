# Social Media — Content Management

Sanity Studio + Flask app for managing AXKAN social content across channels.

## Commands

```bash
cd studio
pip install -r requirements.txt
python app.py           # Launch Flask server
```

## Key Files

- `studio/app.py` — Flask backend (116KB)
- `studio/index.html` — Frontend UI
- `studio/assets/logos/` — Logo variants for social posts
- `PROMPT.md` — Content generation prompts
