# How to Switch Designs Database to Gallery View

## Why Gallery View?

**Gallery view displays images as beautiful cards** - perfect for viewing product designs!

**Table view** = Spreadsheet with tiny thumbnails
**Gallery view** = Large image cards that are easy to see

---

## One-Time Setup (30 seconds)

### For Each Order's Designs Database:

1. **Open the order** in Notion
2. **Scroll to the "Designs" database** (the table inside the order page)
3. **Click "Table"** dropdown (top left of the database)
4. **Click "+ New view"**
5. **Select "Gallery"**
6. **Name it:** "Gallery" (or leave default)
7. **Click "Create"**

**Done!** Images now display as large cards 🎉

---

## Set Gallery as Default View

To make Gallery view open by default:

1. In the Designs database, click the **view dropdown**
2. Select your **Gallery view**
3. Click the **"•••" menu** (next to view name)
4. Click **"Set as default view"**

Now every time you open this order, you'll see Gallery view first!

---

## Customize Gallery Cards (Optional)

Make your gallery even better:

### Show More Info on Cards:

1. In Gallery view, click **"Properties"** (top right)
2. Toggle on properties you want to see:
   - ✅ Tipo (product type)
   - ✅ Requeridos (required quantity)
   - ✅ Contados (counted)
   - ✅ Cajas (boxes)
3. Click outside to close

### Change Card Size:

1. In Gallery view, click **"•••"** (top right)
2. Select **"Card size"**
3. Choose: Small, Medium, or Large
4. **Recommendation:** Large (shows images best!)

### Change Card Preview:

1. Click **"Properties"** (top right)
2. Under **"Card preview"**, select:
   - **Page cover** (shows full image - recommended!)
   - Page content
   - None

## Gallery View Features

**What You'll See:**

- ✅ **Large product images** as card covers
- ✅ **Design name** as title
- ✅ **Editable fields** (Tipo, Requeridos, Contados, Cajas)
- ✅ **Click any card** to open full details

**Benefits:**

- Visual overview of all designs at once
- Easy to spot which products are in the order
- Better for mobile viewing
- Perfect for showing to clients

## Pro Tips

### Filter Designs:

1. Click **"Filter"** (top right)
2. Add rule: e.g., "Contados > 0" (show only counted items)
3. Useful for tracking progress!

### Sort Designs:

1. Click **"Sort"** (top right)
2. Sort by: Design Name, Requeridos, etc.
3. Keeps things organized!

### Search Designs:

1. Use search bar (top right)
2. Find specific products quickly

---

## Switching Between Views

You can have BOTH Table and Gallery views:

- **Table view** - For data entry (typing numbers)
- **Gallery view** - For visual overview (viewing images)

**Toggle between them:**
- Click view dropdown (top left)
- Select "Table" or "Gallery"

---

## Quick Reference

**To create Gallery view:**
- Click "Table" → "+ New view" → "Gallery" → "Create"

**To set as default:**
- View dropdown → "Gallery" → "•••" → "Set as default view"

**To customize cards:**
- "Properties" → Toggle fields on/off
- "•••" → "Card size" → Large

---

## Troubleshooting

**Q: Images not showing in gallery cards?**

A: Make sure:
1. Card preview is set to "Page cover"
2. Google Drive uploaded images successfully
3. Check terminal output for upload confirmation

**Q: Can't see Gallery view option?**

A: Make sure you're:
1. Inside the Designs database (not the Orders database)
2. Clicking the correct view dropdown (top left)

**Q: Want to delete a view?**

A:
1. Switch to the view you want to delete
2. Click "•••" (top right)
3. Click "Delete view"
