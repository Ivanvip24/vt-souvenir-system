# One-Time Setup - Designs Gallery View

## Why This Setup?

After the script creates a new order, the Designs database needs a quick one-time configuration:
- Set **Gallery view as default** (instead of Table)
- Hide the **"Image" property** (images show as covers, no need to show the property)

**Do this once per order** - takes 30 seconds!

---

## Quick Setup (20 seconds)

### Step 1: Create Gallery View

1. **Open your order** in Notion
2. **Scroll to "Designs" database** (the table with your designs)
3. Click **"Table"** dropdown (top left)
4. Click **"+ New view"**
5. Select **"Gallery"**
6. Click **"Create"**

✅ Gallery view created!

---

### Step 2: Set Gallery as Default

1. Make sure you're in **Gallery view** (you just created it)
2. Click the **"•••"** menu (top right of the database)
3. Click **"Set as default view"**

✅ Now every time you open this order, it shows Gallery view first!

---

### Step 3: Done!

✅ **You're done!** Gallery view is ready.

**💡 Using the Image Property:**
- The **"Image" property is visible** - you can click it to upload/change images!
- Images from the script show as **covers** (background)
- You can also **manually upload** images using the Image property
- Both work together perfectly!

**Optional customization:**
- Click **"Properties"** to show/hide any fields you want
- Adjust card size: **"•••"** → **"Card size"** → Large

---

## Result

**Before Setup:**
- Opens in Table view
- Small thumbnails

**After Setup:**
- Opens in **Gallery view** automatically
- Large image cards with covers
- Shows: Design Name, Image (uploadable!), Requeridos, Contados, Printed
- Clean and visual!

---

## Do I Need to Do This for Every Order?

**Yes, but only once per order.**

Each order has its own Designs database, so you need to:
1. Create Gallery view
2. Set as default
3. Hide Image property

**Time:** 30 seconds per order

**Tip:** After doing this a few times, it becomes muscle memory!

---

## Alternative: Template Order (Advanced)

If you create many orders, you can:

1. Set up one order perfectly (Gallery view, hidden properties)
2. **Duplicate that order page** in Notion when creating new orders
3. The duplicated order keeps the view settings!

**But the script is easier** - just run it and do the quick setup!

---

## What's in Each Design?

**Properties:**
- ✅ **Design Name** - Title of the design
- ✅ **Image** - Upload images directly! Also shows as cover/icon
- ✅ **Requeridos** - Required quantity
- ✅ **Contados** - Counted quantity
- ✅ **Printed** - Checkbox to mark when printed

**Removed (not needed):**
- ❌ **"Tipo"** - Not needed per design
- ❌ **"Cajas"** (Boxes) - Tracked at order level (Total Boxes), not per design

---

## Quick Reference

**To set up Gallery view:**
```
1. "Table" → "+ New view" → "Gallery" → "Create"
2. "•••" → "Set as default view"
```

**20 seconds, done!** 🎉

**To upload images in Notion:**
```
Click "Image" property → "Upload" → Select file
```
