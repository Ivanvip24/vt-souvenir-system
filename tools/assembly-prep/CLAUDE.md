# Assembly Prep — Illustrator JSX Scripts

Adobe Illustrator ExtendScript scripts for preparing production files for assembly.

## Usage

Open Adobe Illustrator → File → Scripts → Browse → select the .jsx file.

## Files

- `PREPARACION_ARM.jsx` — Main preparation script
- `backup_armado_preparation.jsx` — Backup version
- `count_elements.jsx` — Count elements in document
- `SETUP_INSTRUCTIONS.md` — Configuration guide
