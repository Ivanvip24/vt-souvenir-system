## Usage Guidelines

### When User Says "Use this element..."

1. **Identify the element** from the four options above
2. **Select appropriate prompt description** from this document
3. **Integrate naturally** into the design composition
4. **Consider placement:**
   - Flor de Tenek: Corner accent, border element, cultural symbol
   - México Letters: Primary destination text or subtitle
   - Pueblos Mágicos Letters: Secondary designation text (below main name)
   - Rehilete Logo: Decorative accent, corner element, festive addition

### Integration Best Practices

- **Scale appropriately** - don't let elements overwhelm main subject
- **Color harmony** - ensure element colors complement overall palette
- **Cultural context** - use elements that make sense for the destination
- **Hierarchy** - common elements should support, not compete with main design
- **Placement logic** - consider traditional positioning (logos in corners, text at bottom, etc.)
