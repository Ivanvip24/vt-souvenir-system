# Core Aspect Ratio Transformation Formula

**Single source of truth for all ratio transformations**

---

## Universal Template (150-200 words)

```
Transform this [current ratio] design to [target ratio] format.

ASPECT RATIO TRANSFORMATION:
[Current] → [Target]
• Composition: [Extend/Reframe/Recompose action]
• Main subject: [Position and size guidance]
• Extended areas: [What fills new space OR what gets removed]
• Layout balance: [How elements redistribute]

Preserve design DNA: [Core elements, colors, style, text, key features]

Style: Match existing [art style description]

CREATE DESIGN
```

---

## Fill-in Guidelines

**Composition actions:**
- Extend: For adding space (square → rectangular)
- Reframe: For removing space (rectangular → square)
- Recompose: For major layout changes

**Main subject:**
- Always specify position (centered/left/right)
- Mention if size should remain constant

**Extended areas:**
- Expanding: What continues (background/elements)
- Cropping: What's safe to remove (edges/margins)

**Design DNA (one line):**
List: Main subject, text, colors, style, key decorative elements
