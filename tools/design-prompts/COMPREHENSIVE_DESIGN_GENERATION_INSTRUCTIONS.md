# 1. SYSTEM OVERVIEW

## Purpose
This system is a comprehensive prompt engineering framework for generating AI-powered souvenir designs. It covers designs for laser-cut MDF products, full-color prints, and various product types including magnets, keychains, bottle openers, ornaments, and more.

## Core Principles

### Design Philosophy
- **Super colorful**: 10+ vibrant colors
- **Irregular organic outlines**: NO rectangular frames unless format-specific
- **Dynamic element interaction**: Elements engage with each other
- **Dense, rich composition**: Typically 8/10 saturation
- **NO black tones or heavy shadows**
- **Celebration and cultural pride**: Festive, welcoming mood

### Three Pillars of Every Design
1. **Creative Concept** - A bold reimagining idea
2. **Spatial Reconstruction** - Complete compositional change
3. **Action Transformation** - Elements doing, not just being

## Production Requirements
All designs must maintain laser-cut MDF compatibility:
- Vector-based artwork
- Line weights: 0.5pt minimum (1pt recommended)
- All elements structurally connected (no floating parts)
- Detail level appropriate for 1/8" MDF material thickness
- Material efficiency target: >70% utilization

# 2. DESIGN WORKFLOW TYPES

## Workflow 1: Design from Scratch
**Purpose:** Create complete designs from just a destination name.

### Required Elements:
- Destination/Place Name
- Style (vintage/modern/minimalist/illustrative/etc.)
- Format (t-shirt/mug/postcard/magnet/etc.)
- Color scheme direction
- Composition type

### Process:
1. Research 3-5 iconic landmarks/symbols
2. Include local flora/fauna
3. Incorporate architectural/historical references
4. Add text element with destination name
5. Apply aesthetic direction matching place character

---

## Workflow 2: Design Based on a Previous Element
**Purpose:** Take a specific unchangeable element and build a complete design around it.

### Fixed Element Types:
- Character illustrations (mascots, animals)
- Logo badges or emblems
- Specific graphic elements (vehicles, objects)
- Brand elements requiring consistency

### Process:
1. Protect fixed element (no modifications)
2. Build environment reflecting destination
3. Add decorative elements with local flavor
4. Create frame/border with regional style
5. Integrate text elements naturally
6. Match style to fixed element's aesthetic

---

## Workflow 3: Generate Variations from Existing Design
**Purpose:** Create TRUE variations that share DNA with originals but are distinctly different.

### Critical Requirements:
- **NOT just color/style changes**
- Must have SPATIAL RECONSTRUCTION
- Elements must perform NEW ACTIONS
- 40-60% recognizability is ideal
- 3-7 new elements added

### Variation Types:
1. Color Variation (Levels 1-6)
2. Style Variation (Levels 3-9)
3. Composition Variation (Levels 2-8)
4. Seasonal Variation (Levels 2-7)
5. Format Variation (Levels 3-6)

---

## Workflow 4: Modify Existing Design
**Purpose:** Make precise surgical edits to existing designs.

### Modification Scope:
- Replace specific elements
- Adjust text content
- Change color palette
- Resize/reposition elements
- Add/remove decorative details

# 3. THE CORE PROMPT FORMULA

## The 150-300 Word Structure (80-90% Success Rate)

```
FORMAT: [Size specification]

CONCEPT: [One-sentence creative hook describing transformation]

SPATIAL RECONSTRUCTION:
[2-3 sentences describing how layout completely changes]

CHARACTER TRANSFORMATION:
[How elements actively change behavior/poses]

NEW ELEMENTS ADDED:
- [5-10 simple items, no details needed]

CHANGES:
- Spatial: [Old structure] → [New structure]
- Characters: [Old pose] → [New action]
- Text: [Old placement] → [New arrangement]
- Style: [If applicable]

CREATE DESIGN
```

## Word Count Discipline
| Section | Target Words |
|---------|--------------|
| Format | 5-10 |
| Concept | 10-15 |
| Spatial | 30-50 |
| Character | 30-50 |
| New Elements | 20-40 |
| Changes | 40-60 |
| **TOTAL** | **150-300** |

## What NOT to Include
- CMYK or hex color codes
- Line weights or thickness measurements
- Material specifications
- Laser cutting details
- Production constraints
- Detailed positioning coordinates
- Font specifications

## What TO Use Instead
- Simple color names ("turquoise ocean", "golden sunset")
- General size relationships ("giant", "tiny", "oversized")
- Creative descriptions ("vintage", "playful", "dramatic")

# 4. COMPOSITION FRAMEWORKS

## Framework 1: Centered Hero
**Hero placement:** Absolute center, 60-80% of design
**Best for:** Strong recognizable landmarks, iconic mascots, badge/medallion styles

**Prompt Language:**
"Centered composition with [HERO] dominating center occupying 70% of design. Supporting [elements] frame left and right edges symmetrically. [TEXT] arcs across top."

---

## Framework 2: Environmental Integration
**Hero placement:** Center or off-center, 35-40%
**Environment:** Equally important at 40-50%
**Best for:** Animals in habitat, cultural scenes, landscape-focused designs

**Prompt Language:**
"Environmental integration composition with [HERO] positioned center-right occupying 35% within [SETTING]. [Background landscape] fills upper third, [foreground elements] anchor bottom."

---

## Framework 3: Diagonal Journey
**Hero placement:** Mid-journey on diagonal path, 30-40%
**Best for:** Adventure/exploration themes, hiking/trail designs, journey narratives

**Prompt Language:**
"Diagonal journey composition with stone trail cutting from lower left to upper right. [HERO] positioned mid-trail occupying 35% suggesting exploration."

---

## Framework 4: Radial Mandala
**Hero placement:** Absolute center, 25-35%
**Structure:** Concentric circles radiating outward
**Best for:** Spiritual/meditative themes, sacred cultural designs, circular formats

**Prompt Language:**
"Radial mandala composition with [HERO] at absolute center occupying 30%. Inner ring of [8 elements] creates first circle, middle ring of [12 elements] forms second layer."

---

## Framework 5: Narrative Scene
**Hero placement:** Left or right third, 35-45%
**Action focus:** Hero doing something specific
**Best for:** Cultural activities, festival scenes, market/vendor designs

**Prompt Language:**
"Narrative scene composition with [HERO] positioned left occupying 40% [performing ACTIVITY]. [Props/objects] on right side support story."

---

## Framework 6: Layered Depth
**Hero placement:** Midground layer, 35-45%
**Structure:** Clear foreground (20%), midground (40%), background (30-40%)
**Best for:** Landscape-heavy designs, atmospheric scenes, mountain/valley compositions

**Prompt Language:**
"Layered depth composition with clear spatial planes. Background [mountains/sky] fills upper 30%, midground [HERO] occupies center 40%, foreground [details] anchors bottom 20%."

---

## Framework 7: Frame-Within-Frame
**Hero placement:** Centered within frame, 35-40%
**Border emphasis:** Thick decorative frame (20-30%)
**Best for:** Formal commemorative designs, ornamental approaches

**Prompt Language:**
"Frame-within-frame composition with [HERO] centered within decorative frame occupying 40% of interior space. Thick ornamental border features [cultural patterns]."

---

## Framework 8: Asymmetric Balance
**Hero placement:** One side, 50-70%
**Counter-balance:** Smaller elements opposite (15-25%)
**Best for:** Dynamic character poses, modern contemporary aesthetics

**Prompt Language:**
"Asymmetric balanced composition with [LARGE HERO] dominating left side occupying 65%. Smaller [counter-elements] balance right side at 20%."

---

## Framework 9: Vertical Stacking
**Hero placement:** Center vertical band, 30-40%
**Structure:** Clear horizontal divisions
**Best for:** Portrait formats, tower/vertical monuments, top-to-bottom narratives

**Prompt Language:**
"Vertical stacking composition with [TEXT] banner across top 15%, [upper background] fills 25%, [HERO] occupies center band 35%, [ground/base] anchors bottom 25%."

---

## Framework 10: Horizontal Spread
**Hero placement:** Center, 30-40%
**Structure:** Left-right flow, panoramic
**Best for:** Landscape formats, journey narratives, wide scene compositions

**Prompt Language:**
"Horizontal spread composition in landscape format with [HERO] centered occupying 35%. [Left scene elements] fill left 25%, [right scene elements] balance right 25%."

# 6. TRANSFORMETER SCALE

## Scale Overview (1-10)

### Levels 1-3: SUBTLE
- Minor tweaks, color shifts, small adjustments
- Same composition maintained
- **90% similarity** to original

### Levels 4-6: MODERATE
- Different palettes, layout changes
- New decorative elements, pose variations
- **60-70% similarity** to original

### Levels 7-9: MAJOR
- Radical style changes, complete restructuring
- Significant new elements, different visual approach
- **30-40% similarity** to original

### Level 10: COMPLETE REDESIGN
- Only subject matter remains
- Entirely new approach and artistic style
- **10% similarity** to original

## Default Levels by Variation Type
| Variation Type | Recommended | Range |
|----------------|-------------|-------|
| Color | 3 | 1-6 |
| Style | 6 | 3-9 |
| Composition | 5 | 2-8 |
| Seasonal | 5 | 4-7 |
| Format | 4 | 3-6 |

## Quick Reference
- **Testing similar options**: Level 2-4
- **Product line expansion**: Level 4-6
- **Major style change**: Level 6-8
- **Complete reimagining**: Level 8-10

## Smart Defaults
- **Default Level**: 5 (moderate change)
- **Professional**: 6-7
- **Conservative**: 3-4
- **Bold**: 7-8

# 7. VARIATION TYPES & SCENARIOS

## Color Variation (Levels 1-6)

### Level 1-2: Subtle Color Shifts
- Adjust hue by 10-30 degrees
- Modify saturation/brightness by 10-20%
- Maintain overall color temperature

### Level 3-4: Moderate Color Changes
- Introduce complementary schemes
- Shift warm to cool (or vice versa)
- Change 2-3 primary colors

### Level 5-6: Major Color Reimagining
- Complete palette swap
- Monochromatic to multicolor
- Dramatic contrast changes

---

## Seasonal Variation (Levels 2-7)

### CRITICAL: Must Be TRUE Variations
Seasonal variations require:
- DECONSTRUCT original into core elements
- REIMAGINE spatial arrangement for seasonal context
- CHANGE element treatments (clothing, poses, activities)
- ADD 3-7 season-specific elements
- TRANSFORM color palette for seasonal mood

### NOT Valid Seasonal Variation:
- Just adding Santa hat to unchanged design
- Placing snowflakes around existing composition
- Color swap without compositional changes

# 8. SPATIAL RECONSTRUCTION PATTERNS

## CORE RULE
**Every variation MUST have spatial reconstruction. No exceptions.**

## Pattern Categories

### 1. Container Transformations
Place the entire design inside something:
- **Snow globe** - Elements float inside sphere
- **Message in bottle** - Design compressed inside glass
- **Coffee cup** - Design floating on coffee surface
- **Open book pages** - Pop-up book concept
- **Puzzle pieces** - Design fragmented across pieces

### 2. Journey/Path Patterns
Create movement through composition:
- **Treasure map** - Dotted path connecting destinations
- **Road/highway** - Elements along winding road
- **Timeline** - Chronological progression
- **Nautilus spiral** - Growing progression
- **Circular wreath** - Elements orbit center

### 3. Perspective Shifts
Change viewing angle completely:
- Front view → Side profile
- Ground level → Aerial/bird's eye
- Flat → 3/4 perspective
- Static → In motion
- 2D flat → 3D layered

### 4. Fragmentation Patterns
Break design into separate pieces:
- **Polaroid photos** - Multiple angled snapshots
- **Postcards** - Overlapping travel cards
- **Grid/mosaic** - Organized tiles
- **Comic panels** - Sequential boxes
- **Floating islands** - Separated land masses

### 5. Shape Transformations
Change overall silhouette:
- Rectangle → Circle/oval
- Rectangle → Triangle/pyramid
- Rectangle → Diamond/rhombus
- Rectangle → Organic natural shape
- Rectangle → Object shape (surfboard, bottle, tree)

### 6. Physics/Logic Inversions
Break normal rules creatively:
- **Upside-down world** - Everything inverted
- **Floating elements** - Weightless arrangement
- **Water flowing up** - Reversed physics
- **Miniature in giant object** - Tiny world concept

## Pattern Selection Matrix
| Original Layout | Best Transformations |
|-----------------|---------------------|
| Horizontal strip | Circular, Vertical tower, Diagonal |
| Centered circle | Scattered fragments, Square grid, Spiral |
| Vertical stack | Horizontal journey, Circular wreath |
| Static portrait | Motion blur, Multiple positions, 3/4 view |
| Flat design | Container concept, Layered depth, Pop-up |

## What is NOT Spatial Reconstruction:
- Simply tilting entire design 15°
- Moving all elements slightly left/right
- Changing only background shape
- Rotating composition as whole
- Scaling everything larger/smaller

# 9. ACTION TRANSFORMATION LIBRARY

## Core Principle
**Static descriptions create static designs. Action creates energy.**

## Movement Actions

### Swimming/Water Actions
- Swimming through (tunnel, coral, waves)
- Diving into (water, scene, container)
- Surfing on (wave, object, current)
- Floating in (bubble, current, air)
- Splashing through (water, puddles)

### Flying/Air Actions
- Flying between (objects, frames, levels)
- Soaring above (landscape, elements)
- Gliding through (clouds, mist, scene)
- Hovering over (specific element)
- Swooping down toward (target)

### Ground Movement
- Running along (path, beach, road)
- Climbing up (ladder, pyramid, tree)
- Jumping between (platforms, frames)
- Dancing around (other elements)
- Sliding down (slope, rainbow, water)

## Interaction Actions

### Holding/Carrying
- Holding (camera, sign, balloon, flag)
- Carrying (luggage, surfboard, basket)
- Lifting (heavy object, another character)
- Balancing (items on head, wings, back)
- Juggling (multiple items)

### Creating/Building
- Arranging (photos, pieces, elements)
- Building (structure, tower, nest)
- Painting (scene, other element)
- Decorating (tree, space, frame)
- Assembling (puzzle, craft, design)

## Expressive Actions

### Greeting/Welcoming
- Waving (with wing, hand, fin)
- Beckoning (come here gesture)
- Pointing at (destination, element)
- Saluting (military, casual, playful)

### Playful/Fun
- Winking at (viewer, other character)
- Taking selfie with (landmark, group)
- Photobombing (jumping into frame)
- Peeking from behind (object, corner)
- Playing (instrument, game, sport)

## Action Description Formula
**Good format:** "[Character] [action verb]ing [object/direction] with [body part/tool]"

**Examples:**
- "Flamingo waving with right wing while holding camera in left"
- "Turtle swimming through coral arch looking back at viewer"
- "Van surfing on giant wave with doors open like wings"
- "Iguana pushing puzzle piece into place with both hands"

## Static Descriptions to AVOID
| Never Write | Always Write |
|-------------|--------------|
| "Positioned at left side" | "Running toward left side" |
| "Located in corner" | "Peeking from corner" |
| "Placed near the top" | "Flying near the top" |
| "Standing in front of" | "Dancing in front of" |
| "Sitting next to" | "Playing guitar next to" |

# 10. PRODUCT TYPE SPECIFICATIONS

## 2:1 Horizontal (10" × 5") - Wall Art/Posters
- Design MUST span FULL 10" WIDTH
- Elements distributed HORIZONTALLY
- Text spans minimum 8" (80% of width)
- NO centered square composition
- Panoramic landscape style emphasis

## 1:1 Square (8" × 8") - Standard Products
- Balanced composition in ALL FOUR DIRECTIONS
- All four corners have visual interest
- Radial, symmetric, or centered focal arrangements
- Equal visual weight in all quadrants
- Works when viewed from any rotation

## 1:2 Vertical (5" × 10") - Tall Formats
- Design MUST use FULL 10" HEIGHT
- Elements STACKED VERTICALLY
- Keep composition within 5" width
- Create visual flow from TOP TO BOTTOM
- Text flows vertically or stacked horizontally

## Bottle Opener (3" × 6")
**CRITICAL - THREE VOID AREAS:**
- TOP VOID: Large circle 1.2" diameter - bottle opening function
- MIDDLE VOIDS: Two small circles 0.3" diameter - mounting holes
- Design elements MUST FLOW AROUND voids
- Text curves ABOVE large top void
- Natural wood background visible

## Magnet (3.5" × 4")
- SIMPLIFIED with BOLD shapes
- HIGH CONTRAST (viewed from 5-10 feet)
- NO fine details smaller than 0.125"
- Text minimum 0.5" tall with 2-3pt bold outlines
- Limit to 2-4 words maximum

## Bookmark (2.5" × 7")
- Ultra NARROW VERTICAL format
- Tassel hole: 0.1875" diameter at top
- Elements STACKED TOP TO BOTTOM
- HORIZONTAL REINFORCEMENTS required
- All vertical elements connect at multiple points

## Ornament (4" Diameter)
- CIRCULAR composition (radial or concentric)
- Hanging hole: 0.25" at top center
- All elements connect to outer circle
- Text curves along circular path
- Design readable when hanging vertically

## Coaster (3.5" × 3.5")
- Small PERFECT SQUARE
- FUNCTIONAL surface (drinks placed on top)
- Center may be covered by glass
- Important elements around perimeter
- Simplified for small viewing size

## Keychain (1.5-2.5")
- ULTRA SMALL - MAXIMUM SIMPLIFICATION
- BOLD shapes only - NO fine details
- Text minimum 0.5" with 3pt BOLD outlines
- Limit to 1-2 words OR iconic symbol only
- Design recognizable at arm's length

# 12. SAFETY & CONTENT GUIDELINES

## Content to ABSOLUTELY AVOID

### Child Safety
- NO children in any design elements
- NO imagery that could be interpreted as exploitative

### Dangerous Activities
- NO weapons (even historical/cultural)
- NO drug-related imagery (even ceremonial)
- NO self-harm symbolism

### Violent/Gory Content
- NO blood or injury depictions
- NO violent historical scenes (conquest, battles)
- NO animal cruelty or hunting

### Harassment/Discrimination
- NO suggestions that any group is inferior/superior
- NO dehumanizing comparisons to animals
- NO stereotypes that demean or mock

## High-Risk Elements for Mexican Designs

### Symbols That May Block:
- Revolutionary/political symbols (Che, weapons)
- Conquest scenes (Spanish vs. Indigenous)
- Stereotypes (sleeping under cactus, drunk depictions)
- Religious imagery in irreverent contexts
- "Propaganda poster" aesthetic

### Safe Alternatives:
- Focus on celebrations, crafts, nature, architecture
- Use vintage travel poster (not propaganda poster)
- Specific tribal names (Seri, Zapotec) not vague "indigenous"
- Celebratory cultural pride, not exploitation

## Language Patterns That Work

**Safe Descriptive Patterns:**
- "Traditional [tribe name] geometric patterns inspired by authentic basketry"
- "Vibrant celebration of [region] culture through [specific craft]"
- "Colonial architecture from the 1700s featuring adobe arches"
- "Day of the Dead celebration with colorful decorated skulls, marigold flowers"

**Safe Style References:**
- "Vintage travel poster aesthetic"
- "Mid-century modern illustration style"
- "Art Deco geometric design"
- "Contemporary Mexican graphic design"
- "Folk art illustration style"

**AVOID Style References:**
- "Propaganda poster"
- "Revolutionary art style"
- "Nationalist design"
- "Political muralism"

# 13. VALIDATION CHECKLISTS

```
□ Canvas is EXACTLY 10" WIDE × 5" TALL (2:1 ratio)
□ Design MUST span FULL 10" WIDTH
□ Elements distributed HORIZONTALLY
□ Text spans minimum 8" (80% of width)
□ NO centered square composition
□ Background fills ENTIRE rectangle
```

## Pre-Generation Verification (1:1 Square)
```
□ Canvas is EXACTLY 8" × 8" (perfect square)
□ Balanced composition in ALL FOUR DIRECTIONS
□ All four corners have visual interest
□ Use radial, symmetric, or centered arrangements
□ Equal visual weight in all quadrants
□ NO elongated horizontal or vertical bias
```

## Pre-Generation Verification (Bottle Opener)
```
□ Canvas is EXACTLY 3" × 6" (1:2 vertical)
□ Outer shape is ROUNDED RECTANGLE
□ Background is NATURAL WOOD TEXTURE
□ TOP VOID: 1.2" diameter - EMPTY
□ MIDDLE VOIDS: Two 0.3" circles - EMPTY
□ Design flows AROUND void areas
□ Text curves ABOVE large top void
```

## True Variation Validation
```
□ Creative concept stated in ONE sentence
□ Spatial structure completely DIFFERENT
□ Characters/elements doing NEW ACTIONS
□ 5-10 new elements added
□ 40-60% recognizability (not 85% similar)
□ Could NOT overlay with original
□ Prompt is 150-300 words total
□ 80% focuses on transformation
□ NO technical specs (CMYK, line weights)
□ Clear before → after changes listed
```

## Quality Checklist (Before Submitting)
```
□ Composition framework named
□ Hero element 30-80% (appropriate to framework)
□ 5-10 supporting elements listed
□ Decoration density specified (6/10, 8/10, 10/10)

□ Style descriptor from library used
□ Color palette: 4-6 simple names (NO CMYK)
□ NO technical specs
□ Edge treatment specified (organic irregular default)
□ Production note at end
□ Ends with "CREATE DESIGN"
□ Total 200-350 words
```

## Gemini Safety Checklist
```
□ No weapons, even historical/ceremonial
□ No violent historical scenes
□ No stereotypes suggesting inferiority
□ No political/revolutionary symbols
□ No comparisons of people to animals
□ No sacred religious imagery commercially
```
