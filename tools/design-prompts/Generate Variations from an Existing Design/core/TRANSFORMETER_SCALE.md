# TRANSFORMETER Scale
*The universal scale for controlling variation intensity*

## Scale Overview

**1-10 scale** that controls variation intensity across all variation types:

### Levels 1-3: SUBTLE
- Minor tweaks
- Color shifts
- Small adjustments
- Same composition
- 90% similarity to original

### Levels 4-6: MODERATE
- Different palettes
- Layout changes
- New decorative elements
- Pose variations
- 60-70% similarity to original

### Levels 7-9: MAJOR
- Radical style changes
- Complete restructuring
- Significant new elements
- Different visual approach
- 30-40% similarity to original

### Level 10: COMPLETE REDESIGN
- Only subject matter remains
- Entirely new approach
- Different artistic style
- New concept
- 10% similarity to original

## Default Levels by Type

| Variation Type | Recommended Level | Range |
|---------------|-------------------|-------|
| Color         | 3                 | 1-6   |
| Style         | 6                 | 3-9   |
| Composition   | 5                 | 2-8   |
| Seasonal      | 5                 | 4-7   |
| Format        | 4                 | 3-6   |

## Quick Reference

**Testing similar options**: Level 2-4
**Product line expansion**: Level 4-6
**Major style change**: Level 6-8
**Complete reimagining**: Level 8-10

## Smart Defaults

If not specified, system uses:
- **Default Level**: 5 (moderate change)
- **Professional**: 6-7
- **Conservative**: 3-4
- **Bold**: 7-8
