# Standard Variation Workflow
*Single source of truth for the workflow process*

## 6-Step Workflow

### 1. **PREPARE**
- Review original design
- Define goals
- Choose variation type
- Select TRANSFORMETER level

### 2. **GENERATE**
- Use creation-agent to create 2-5 variations
- Agent combines generation + production feasibility

### 3. **VALIDATE**
- Use validation-agent to check quality
- Agent combines validation + family coherence

### 4. **ITERATE** (if needed)
- Adjust based on validation feedback
- Regenerate specific variations

### 5. **FINALIZE**
- Document variations
- Update index
- Prepare for production

### 6. **DELIVER**
- Export final prompts
- Archive documentation
- Hand off to production team

## Quick Commands

For fast workflow, use:
```
QUICK_VARIATION: "[Destination], [Theme], [Level]"
```

Example:
```
QUICK_VARIATION: "Hermosillo, Christmas, 6"
```
