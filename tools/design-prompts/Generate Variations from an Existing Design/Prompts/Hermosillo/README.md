## Gemini Safety Optimization

These prompts have been optimized to avoid common Gemini blocks by:
- Avoiding multiple indigenous tribal references
- Removing industrial/mining imagery
- Simplifying cultural pattern descriptions
- Using "traditional basketweaving" instead of specific tribal ownership
- Removing "mission" colonial references
- Focusing on nature elements (always safe)
- Using approved style descriptors ("Art Deco", "vibrant collage")
- Clear celebratory, tourist-friendly tone
