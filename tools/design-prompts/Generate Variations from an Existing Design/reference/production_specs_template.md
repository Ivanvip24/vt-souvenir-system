# Production Specifications Template

**Purpose**: Copy-paste technical specifications for laser-cut MDF souvenir designs. Reduces prompt writing time by providing standard production requirements.

**Usage**: Copy the appropriate template section and paste into Section 4 of your prompt. Adjust dimensions and specific colors as needed.

---

## Standard 2:1 Horizontal Format

```
**Dimensions**: 10" wide × 5" tall (2:1 ratio)
**Material**: 1/8" (3mm) laser-cut MDF
**Finish**: Full-color UV printing (300 DPI minimum)
**File Format**: Vector (Adobe Illustrator .ai or .eps)
**Color Mode**: CMYK for printing accuracy

**Laser-Cut Requirements**:
• All primary outlines: 2pt stroke weight (black)
• Secondary element outlines: 1.5pt minimum stroke weight
• All elements structurally connected (no floating pieces)
• No elements smaller than 0.25" in any dimension
• Internal pattern/texture details rendered for UV printing only, not laser cutting
• Cut path: Single continuous outline around entire design perimeter
• Connection points: All elements must touch adjacent elements or main frame

**Typography**:
• Primary text: Custom decorative bold letterforms with 2pt outline
• Secondary text: Decorative sans-serif or geometric font, [X]% of canvas height
• Tertiary text: Classic serif or complementary font, [X]% of canvas height
• All text must have sufficient contrast with background for readability

**Production Timeline**:
• Design finalization: 2-3 days
• Vector file preparation: 1-2 days
• Test cut/print: 1 day
• Production run: Same-day capability for up to 100 units
```

## Quick Reference: Line Weight Standards

**By Element Type**:
- Main letterforms: **2pt** (maximum visibility and cutting strength)
- Large decorative elements (cacti, landmarks): **1.5-2pt**
- Medium elements (animals, plants): **1.5pt minimum**
- Small details/accents: **1pt minimum** (only for elements >0.5" in size)
- Border/frame lines: **1.5-2pt**
- Connecting elements: **1.5pt minimum** (structural requirement)

**By Format Size**:
- Large (>8" in any dimension): 1.5pt minimum acceptable
- Medium (4-8"): 1.5-2pt recommended
- Small (<4"): 2pt strongly recommended
- Magnet/Keychain: 2-3pt required

## Quick Reference: Typography Guidelines

**Font Selection**:
- **Primary text** (destination name): Bold decorative, hand-drawn style, or custom lettering
- **Secondary text** (state/region): Complementary decorative or clean sans-serif
- **Tertiary text** (additional info): Classic serif or simple sans-serif

**Text Sizing**:
- **Large format** (8-10" wide): Primary 30-40% of height, Secondary 15-20%, Tertiary 10-12%
- **Medium format** (4-6" wide): Primary 35-45% of height, Secondary 18-22%, Tertiary 12-15%
- **Small format** (2-4" wide): Primary 40-50% of height, limit secondary, avoid tertiary

**Readability Requirements**:
- Minimum letter height: 0.375" for distant viewing (wall art, magnets)
- Minimum letter height: 0.5" for small formats (bookmarks, keychains)
- Outline stroke: 2pt minimum for definition against busy backgrounds
- Contrast: Text must have >70% contrast with background/fill colors

**Curved Text**:
- For circular/medallion designs: Text follows radius perpendicular path
- Letter spacing: Slightly increased for curved paths (110-120% of normal)
- Readability check: Text readable when rotated to reading angle

## Quick Reference: Structural Integrity Rules

**Connection Requirements**:
1. **All elements must physically connect** to at least one other element
2. **No floating pieces** - every shape touches the main design or frame
3. **Minimum connection width**: 0.125" where elements touch
4. **Bridge connections**: Use decorative elements (vines, lines, patterns) to connect disparate parts

**Format-Specific Stability**:
- **Vertical formats**: Add horizontal crossbars or background elements for stability
- **Narrow formats** (bookmarks): Multiple connection points along length
- **Circular formats**: Radial elements must have at least 3 connection points to edge
- **Letter-based designs**: Letters touch at baseline or overlap at edges

**Testing Structural Integrity**:
- All parts must be accessible by single laser cut path (no need to reposition)
- Design should feel stable when held (no weak connection points)
- Thin vertical elements need horizontal reinforcement
- Islands must connect via bridges

## Troubleshooting Common Issues

**Issue**: AI creates floating elements
**Solution**: Add to specs: "All elements must physically connect. Use decorative bridges (vines, lines, geometric connectors) to join isolated parts."

**Issue**: Details too fine for laser cutting
**Solution**: Specify: "Fine details (textures, patterns, small features) are UV print only. All cut lines minimum 1.5pt weight."

**Issue**: Text unreadable on busy backgrounds
**Solution**: Add: "Text requires 2pt outline stroke in contrasting color for definition. Ensure 70%+ contrast with background."

**Issue**: Vertical design too fragile
**Solution**: Specify: "Add horizontal reinforcement elements. All vertical pieces must connect to horizontal crossbars."

**Issue**: Color too complex for production
**Solution**: Use CMYK specs from palette guide above. Limit to 6-8 distinct colors per design.
