# Speed-Optimized Design Creation System

## Overview
This system creates original destination souvenir designs in **6-8 minutes** using composition-first methodology and streamlined prompts (200-350 words).

**Target Performance:**
- Reference analysis: 2 minutes
- Element selection: 2 minutes (using libraries)
- Composition selection: 1 minute
- Prompt writing: 3-4 minutes
- **TOTAL: 6-8 minutes per design**

---

## Core Philosophy

### Composition-First (Not Detail-First)
```
OLD: Character → Cultural Context → Composition → Verify → Finalize
NEW: Composition Framework → Hero Element → Supporting Elements → Style
```

Start with spatial structure, then fill it in.

### 200-350 Word Prompts (Not 400+ Lines)
- Concise but complete
- Simple color names (not CMYK)
- Proven descriptors (not custom essays)
- Structure over specification

### Library-Based (Not Research-Each-Time)
- Pre-built element libraries
- Copy-paste descriptions
- Proven composition patterns
- Standard style descriptors

---

## The 4-Step Process

### STEP 1: Analyze & Choose Framework (2-3 minutes)

#### Analyze the Source Element
Quick assessment:
- **Hero element:** What's the protagonist? (character, landmark, animal)
- **Visual DNA:** What makes it recognizable? (shape, color, key detail)
- **Style signals:** What aesthetic? (cartoon, realistic, vintage, modern)
- **Composition opportunity:** How should space be organized?

#### Choose Compositional Framework
Pick ONE from library:
- **Centered Hero:** Character/landmark dominates center (60-80%)
- **Environmental Integration:** Element in natural setting (35-40% element)
- **Narrative Scene:** Element doing something, story-driven
- **Radial Mandala:** Element centered in circular composition
- **Diagonal Journey:** Path/trail composition with movement
- **Layered Depth:** Strong foreground/midground/background
- **Frame-Within-Frame:** Border/arch containing scene

### STEP 2: Select Elements (2 minutes)

#### Identify Sacred Elements (CAN'T Remove)
2-3 elements essential to design identity:
- The hero element itself
- 1-2 defining features/props
- Destination name (prominent)

#### Copy 5-10 Supporting Elements
From regional libraries:
- **Flora:** 3-4 native plants
- **Fauna:** 1-2 supporting animals (if applicable)
- **Cultural:** 2-3 objects/patterns/architecture
- **Environmental:** Landscape features

**Time-saver:** Don't research from scratch, use library descriptions

### STEP 3: Write Prompt (3-4 minutes)

Use the standard formula (see PROMPT_TEMPLATE.md):
1. Format (1 line)
2. Subject (1 sentence)
3. Composition Structure (2-3 sentences)
4. Protagonist Element (1 paragraph)
5. Supporting Elements (simple list, 5-10 items)
6. Decorative Layer (1 paragraph)
7. Text Integration (1 paragraph)
8. Style & Aesthetic (2-3 sentences)
9. Color Palette (1 sentence, 4-6 colors)
10. Edge Treatment (1 sentence)
11. Production (1 sentence)
12. "CREATE DESIGN"

**Target:** 200-350 words total

### STEP 4: Generate & Iterate

Submit prompt to AI image generator. If revision needed, adjust composition or elements—don't rewrite entire prompt.

## Key Principles

### Visual Hierarchy (Sacred Rule)
- **Hero element:** 60-80% visual weight (if landmark/character-focused)
- **Or balanced:** 30-40% hero + 30-40% environment (if scene-focused)
- **Supporting cast:** 15-25% (context elements)
- **Decoration:** 5-15% (fills space, adds richness)

### Text Prominence (Non-Negotiable)
- **Primary text:** 15-25% of design height
- Must be readable from 3 feet away
- Integrated into design, not floating
- Users WILL complain if too small

### Edge Treatment Default
- **Default:** Organic irregular shapes
- Avoid rectangles unless format requires
- Border follows natural contours of elements
- No straight edges or 90° corners

### Uniqueness Requirement
**CRITICAL:** Each design is for a new client order
- Never repeat compositions even for same destination
- Change protagonist angle/pose/action
- Different supporting element selection
- Different decorative approach
- Different edge treatment

Even for same location, create unique variations.

## Style Specification Shortcuts

Don't write style from scratch—use proven descriptors:

**Whimsical Cartoon:**
"Playful cartoon illustration with bold black outlines, flat vibrant colors, friendly character design, cheerful approachable mood"

**Realistic Illustrative:**
"Realistic illustrative style with photographic quality details, natural lighting, dimensional depth, authentic textures, documentary mood"

**Vintage Travel Poster:**
"Vintage travel poster aesthetic, bold simplified shapes, limited color palette, hand-lettered typography, nostalgic 1950s National Parks charm"

**Modern Flat Design:**
"Contemporary flat design, clean vector shapes, vibrant colors, minimal shading, geometric patterns, crisp edges"

**Vintage Naturalist:**
"Vintage scientific botanical illustration style, detailed specimens, sepia tones, scholarly aesthetic, museum field guide quality"

**Decorative Folk Art:**
"Traditional folk art style, intricate pattern fills, ornamental borders, nature motifs, rich decoration, handcrafted feel"

**Mystical Atmospheric:**
"Mystical enchanted illustration with atmospheric effects, magical lighting, spiritual mood, ethereal quality, reverent tone"

Just name the style + 2-3 key traits = enough

## Composition Framework Quick Reference

### 1. Centered Hero (Most Common)
- Hero dominates center: 60-80%
- Supporting elements frame around edges
- Text arcs top or bottom
- Balanced symmetrical feel

**Best for:** Strong landmarks, iconic characters, monuments

### 2. Environmental Integration
- Hero in natural setting: 35-40%
- Environment equally important: 35-40%
- Hero appears "in" setting not "on" it
- Layered depth (foreground/mid/background)

**Best for:** Animals in habitat, cultural scenes, nature-focused

### 3. Diagonal Journey
- Path/trail cutting diagonally
- Hero on path: 30-40%
- Implies movement/exploration
- Dynamic asymmetric energy

**Best for:** Adventure themes, hiking, exploration narratives

### 4. Radial Mandala
- Perfect circular symmetry
- Hero at absolute center: 25-35%
- Concentric rings radiating outward
- Sacred geometry aesthetic

**Best for:** Spiritual themes, meditative designs, symmetrical beauty

### 5. Narrative Scene
- Hero performing action: 35-45%
- Props/environment support story
- Clear activity/interaction
- Storytelling focus

**Best for:** Cultural activities, festivals, character interactions

### 6. Layered Depth
- Strong foreground (20%)
- Midground hero (40%)
- Background setting (40%)
- Clear spatial planes

**Best for:** Landscapes, scenes with depth, atmospheric designs

### 7. Frame-Within-Frame
- Decorative border frames scene
- Hero centered in frame: 35-40%
- Border tells secondary story
- Contained medallion feel

**Best for:** Formal designs, badge-style, ornamental approaches

## Common Pitfalls to Avoid

### ❌ The Description Trap
Writing literal photographic descriptions:
"A turtle with green shell swimming in blue water near coral"
- Too literal, lacks structure
- No composition direction
- No style guidance

### ✅ The Compositional Approach
"Centered sea turtle swimming diagonally upward occupies 70% of composition. Coral reef arc frames bottom. Decorative water ripples fill background. Realistic illustrative style with natural lighting."

---

### ❌ The Everything Kitchen Sink
Listing 40+ elements with detailed specs for each
- Overwhelming, confusing
- AI can't prioritize
- Usually fails

### ✅ The Focused Hierarchy
"Protagonist: Saguaro cactus (70%). Supporting: Desert sunset, mountains, roadrunner (25%). Decoration: Small flowers, geometric patterns (5%)."

---

### ❌ The Technical Overload
"CMYK 85/50/0/0 for sky, line weight 1.5pt, Helvetica Bold 48pt..."
- May trigger content blocks
- Overly prescriptive
- Reduces creative quality

### ✅ The Creative Direction
"Bright turquoise sky, vibrant magenta flowers, bold readable typography, decorative pattern fills"

## Decoration Density Levels

Specify clearly in prompt:

**6/10 Decoration (Moderate):**
- 20-30 decorative elements
- 30-40% negative space preserved
- Clean, uncluttered feel
- Breathing room maintained

**8/10 Decoration (Abundant):**
- 40-60 decorative elements
- 10-20% negative space
- Multiple decoration systems layered
- Richly filled but not overwhelming

**10/10 Decoration (Maximal):**
- 60-80+ decorative elements
- <10% negative space
- "Horror vacui" - fear of empty space
- Every corner densely filled

In prompt:
"Decoration level 8/10: Abundant floral elements, geometric border patterns, scattered wildlife accents filling most negative space."

---

## Production Consciousness

Include at END of every prompt (single sentence):

**For Laser-Cut:**
"All elements structurally connected for laser-cut MDF production with appropriate line weights."

**For Print:**
"High-resolution detail suitable for full-color print production."

**For Both:**
"All elements structurally connected, suitable for laser-cut MDF or full-color print."

Don't elaborate in creative sections—keep technical at end.

## Formula Variations by Format

**Square (1:1) - Most Common:**
- Centered or radial works best
- Balanced decoration all edges
- Text curves along bottom or circles center

**Portrait (2:3, 4:5) - Vertical:**
- Vertical stacking natural
- Tower/ascending compositions
- Text at top or bottom banner

**Landscape (3:2, 5:4) - Horizontal:**
- Horizontal journey/path compositions
- Left-to-right narrative flow
- Text as banner across design

**Circle:**
- Radial mandala composition
- Concentric rings
- Text curves along perimeter
